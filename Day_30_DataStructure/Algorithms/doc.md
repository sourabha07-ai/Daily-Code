# C++ Algorithms (Explained Like a Tutor)

## The `<algorithms>` library in C++ provides ready-made functions for common operations like

- [x] Sorting
- [x] Searching
- [x] Counting
- [x] Reversing
- [x] Finding minimum/maximum
- [x] Removing duplicates
- [x] Rotating elements
- [x] And much more...

Instead of writing these algorithms yourself, you can use the STL versions, which are efficient and well-tested

## Common Algorithms Summary

| Algorithm | Purpose |
| --------- | ------- |
| `sort()` | Sort elements |
| `reverse()` | Reverse elements |
| `find()` | Find a value |
| `count()` | Count occurrences |
| `max_element()` | Largest element |
| `min_element()` | Smallest element |
| `binary_search()` | Search in sorted data |
| `lower_bound()` | First valid position |
| `upper_bound()` | Position after last occurrence |
| `swap()` | Exchange values |
| `rotate()` | Rotate elements |
| `unique()` | Remove consecutive duplicates |
| `fill()` | Fill with a value |
| `copy()` | Copy elements |
| `accumulate()` | Sum elements |
