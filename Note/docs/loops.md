# Loops in C++

Loops help you repeat code until a condition changes.

## for loop

```cpp
for (int i = 0; i < 5; i++) {
    std::cout << i << "\n";
}
```

## while loop

```cpp
int i = 0;
while (i < 5) {
    std::cout << i << "\n";
    i++;
}
```

## do-while loop

```cpp
do {
    std::cout << "Execute at least once";
} while (false);
```
