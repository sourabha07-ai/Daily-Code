# Variables in C++

Variables store data that your program can use and modify.

## Common types

- int: whole numbers
- float: single-precision decimals
- double: double-precision decimals
- char: single characters
- bool: true or false

```cpp
int age = 21;
double price = 19.99;
char grade = 'A';
bool isActive = true;
```

## Scope

A variable's scope determines where it can be used. Variables declared inside a function are local to that function.
