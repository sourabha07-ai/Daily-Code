# Set in C++

std::set stores unique values in sorted order.

```cpp
#include <set>

std::set<int> values = {3, 1, 2, 2};
for (int x : values) {
    std::cout << x << " ";
}
```

## Benefits

- Avoids duplicates
- Keeps data ordered
- Useful for membership checks
