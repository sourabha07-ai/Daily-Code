# Getting Started with C++

C++ is a powerful general-purpose programming language used for system software, game development, embedded systems, and high-performance applications.

## Install a compiler

A common setup is:

- GCC or MinGW on Windows
- g++ on Linux or macOS

## Your first program

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, C++!";
    return 0;
}
```

## Tips

- Use meaningful variable names.
- Keep code readable and properly indented.
- Compile often to catch errors early.
