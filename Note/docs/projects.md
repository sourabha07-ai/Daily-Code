# C++ Projects to Practice

These mini-projects help you apply the concepts you learn.

## Beginner ideas

- Calculator
- Number guessing game
- Student grade tracker
- To-do list using vectors

## Intermediate ideas

- File-based contact manager
- Quiz application
- Expense tracker
- Maze solver using stacks

## Practice tips

- Build one feature at a time.
- Test your code carefully.
- Refactor when you see repetition.
